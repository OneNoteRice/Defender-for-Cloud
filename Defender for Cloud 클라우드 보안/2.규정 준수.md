Defender for Cloud 활성화시 다음 표준이 활성화

Azure : Microsoft 클라우드 보안 벤치마크 (MCSB)
AWS: AWS 기본 값 및 Microsoft 클라우드 보안 벤치마크 (MCSB)
GCP: GCP 기본 값 및 Microsoft 클라우드 보안 벤치마크 (MCSB)
